## 01. BRAND STORY

### Brand Essence
Freddie Kashawan adalah personal brand dari seorang entrepreneur yang menggunakan AI di bisnis nyata, lalu mengubah pengalaman itu menjadi konten, kelas, dan sistem yang bisa direplikasi oleh orang lain.

### Core Promise
> "Stop pakai AI random. Mulai pakai AI sebagai sistem."

AI harus jadi sistem yang menciptakan leverage — bukan kumpulan tools yang dipakai acak. Freddie mengajarkan cara membangun sistem AI yang compound: makin lama makin powerful, bukan makin rumit.

### Brand Pillars

## 02. LOGO SYSTEM

### Anatomy
- **Monogram Mark:** Huruf FK italic condensed bold berwarna **Brand Blue #2563FF**. Didesain miring untuk mencerminkan energy, movement, dan forward-thinking mindset.
- **Wordmark:** FREDDIE KASHAWAN dalam font condensed bold semua kapital. Dua baris sejajar menciptakan visual block yang solid dan authoritative.
- **Relasi:** Monogram di kiri sebagai anchor visual. Wordmark di kanan sebagai identitas penuh. Keduanya selalu digunakan bersama kecuali dalam format ikon/favicon.

### Logo Variants

| Variant | Gunakan Ketika | Background |
|---------|---------------|------------|
| Dark (FK putih + wordmark putih) | Background gelap | Navy #0A1628, Black #0A0C14 |
| Light (FK biru + wordmark hitam) | Background terang | White #FFFFFF, Off-White #EEF2FF |
| Monogram FK saja | Favicon, app icon, stiker kecil | Any |

### Minimum Size

| Medium | Min Width |
|--------|-----------|
| Digital / Web | 160px |
| Print | 40mm |
| Monogram Only | 24px / 6mm |
| Favicon | 32 × 32px |

### Clear Space
Minimum = tinggi huruf kapital (cap-height) logo di semua sisi.

### ✓ LAKUKAN
- Gunakan logo dark di background gelap (navy, black)
- Gunakan logo light di background terang (white, off-white)
- Pertahankan proporsi asli logo selalu
- Pastikan kontras minimum 4.5:1 terpenuhi
- Gunakan monogram FK saja untuk favicon dan ikon app

### ✗ JANGAN LAKUKAN
- Jangan ubah warna FK dari Brand Blue #2563FF
- Jangan stretch, skew, atau rotate logo
- Jangan tempatkan di background busy/ramai
- Jangan tambahkan shadow, outline, atau efek apapun
- Jangan recreate logo secara manual — selalu gunakan file asli

## 05. BRAND ELEMENTS

### Corner Radius

| Value | Gunakan Untuk |
|-------|--------------|
| 0px | Dividers, bars, sharp elements |
| 4px | Tags, badges, small elements |
| 8px | Buttons, inputs, small cards |
| 12px | Cards, sections, modal dialogs |

### Graphic Elements
- **Blue Radial Blob:** Digunakan di background hero dan dark sections. `radial-gradient(circle, rgba(37,99,255,0.2) 0%, transparent 70%)`
- **Left Accent Bar:** `border-left: 3px solid #2563FF` — untuk quote blocks dan callouts
- **Right Edge Line:** Vertical gradient line kanan frame — untuk cover dan hero sections

### Button System

| Variant | Style |
|---------|-------|
| Primary | `bg: #2563FF · color: white · radius: 8px` |
| Secondary | `bg: transparent · border: 2px solid #2563FF · color: #2563FF` |
| Dark | `bg: #0A1628 · color: white · radius: 8px` |
| Ghost | `bg: transparent · border: 1.5px solid rgba(255,255,255,0.25) · color: white` |

### Icons
Gunakan **Tabler Icons (Outline style)** — stroke 1.5px. Warna: Brand Blue #2563FF (light bg) / Sky Blue #60A5FA (dark bg). Ukuran: 20px inline, 24px dekoratif.

### Writing Style Rules

| Element | Aturan | Contoh |
|---------|--------|--------|
| Kalimat | Pendek. Fragment diizinkan. Ritme cepat. | "Bukan tools. Sistemnya." |
| Pembuka | Langsung ke inti. Tidak ada basa-basi. | "Ini yang salah dari cara lo pakai AI." |
| Pertanyaan | Retoris untuk engagement | "Kenapa 90% bisnis gagal dengan AI?" |
| Data | Gunakan angka konkret selalu | "10 tahun bisnis. 2 tahun AI. 5 brand aktif." |
| CTA | Direct, specific, action-oriented | "Klik link di bio. Chat sekarang." |
